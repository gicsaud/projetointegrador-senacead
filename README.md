## Funcionalidades

- Tela de cadastro das entidades: Pessoa Física, Pessoa Jurídica, Aluno, Professor e Fornecedor.
- Tela de login (universal para as entidades).
- Página de Listagem de usuários.
- Página de edição de usuários.
- Dashboard (homepage).
